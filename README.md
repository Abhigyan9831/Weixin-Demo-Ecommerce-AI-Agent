# Weixin-Demo-Ecommerce-AI-Agent

## Introduction

A Multilingual E-commerce AI Assistant with Sentiment Analysis, Product Search, Offer Engine & Customer Memory-- This project is a prototype of an AI-powered customer service agent designed for WeChat, Weixin, for e-commerce platforms. It has custom inbuilt tools handles customer reviews, provides personalized offers, fetches product details, and maintains long-term customer interaction history.
